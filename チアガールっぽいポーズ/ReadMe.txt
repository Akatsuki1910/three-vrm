「チアガールっぽいポーズ」をDLして頂きありがとうございます。
お気に入りの娘にポーズしてもらっちゃいましょう！
各ポーズファイルはサンプル画像と対応しております。

---------------------------------------------------------------------------

このポーズは、ままま様の「ままま式あぴミク」モデルを用いて制作しました。
他のモデルの場合、適宜修正が必要となるかもしれません。
ですので申し訳ありませんが、修正は各自でお願いしますorz

またあぴミクモデルには「上半身2」「右or左腕捩・右or左手捩」
などのボーンが入っており、それらを使用してポーズを作りました。
他のモデルでの使用時にはそれらのボーンを追加するなどしてご利用下さい。

---------------------------------------------------------------------------

物理演算を「演算しない」に設定してご使用下さい。
※物理演算「常に演算」では、スカートや髪の毛などが破綻する場合があります。
